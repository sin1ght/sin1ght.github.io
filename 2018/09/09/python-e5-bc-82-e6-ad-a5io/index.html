Python异步IO

Veröffentlicht am 2018-09-09 | Edited on 2019-02-24 | in WEB前端/后端

asyncio是Python 3.4版本引入的标准库，直接内置了对异步IO的支持。 asyncio的编程模型就是一个消息循环。我们从asyncio模块中直接获取一个EventLoop的引用，然后把需要执行的协程扔到EventLoop中执行，就实现了异步IO。

1.@ asyncio.coroutine/yield from

  • `@asyncio.coroutine`把一个函数标记为协程

  • yield from针对耗时的操作进行挂起，函数让出控制权

    import time
    import asyncio
    import random

now = lambda : time.time()

@asyncio.coroutine
def do_some_work(x):
    print("{} start".format(x))
    yield from asyncio.sleep(2)
    print("{} end".format(x))


start = now()

loop = asyncio.get_event_loop()
tasks = [asyncio.ensure_future(do_some_work(i)) for i in range(10)]

loop.run_until_complete(asyncio.wait(tasks))
print("Time:",now()-start)

2.async/await

  • async代替`@asyncio.coroutine`

  • await代替yield from

    import time
    import asyncio
    import random

now = lambda : time.time()


async def do_some_work(x):
    print("{} start".format(x))
    await asyncio.sleep(2)
    print("{} end".format(x))


start = now()

loop = asyncio.get_event_loop()
tasks = [asyncio.ensure_future(do_some_work(i)) for i in range(10)]

loop.run_until_complete(asyncio.wait(tasks))
print("Time:",now()-start)

3.aiohttp

import asyncio
import aiohttp


async def task(url):
        print('start {}'.format(url))
        async with aiohttp.ClientSession() as session:
            async with session.get(url) as resp:
                content=await resp.text()
                print('{1} bytes from {0}'.format(url,len(content)))

urls=['https://github.com/','https://www.baidu.com/','https://www.python.org/']

event_loop=asyncio.get_event_loop()

tasks=[asyncio.ensure_future(task(i)) for i in urls]

event_loop.run_until_complete(asyncio.wait(tasks))
